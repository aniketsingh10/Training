Javascript Assignment 1


Task 1
Accept first name and last name and when the button is clicked, display full name. In the full name, the first letter in first name and last name should be capitalized even if the names are entered in small letters.


Task 2
Accept a string, a word to replace and the new word. When the button is clicked replace the stated word in the string by the new word and display the changed string.


Task 3
Accept a word. When the button is clicked display the word with reversed characters. e.g. The word 'Test' should be displayed as 'tseT'.


Task 4
Accept a fractional number and the number of digits to displayed after decimal. When the button is clicked, round a number to a specified amount of digits and display.


Task 5
Display the current day and time in the following format: Today is : Tuesday. Current time is : 10 PM : 30 : 38
